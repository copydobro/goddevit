# Позиционирование агентства
> v2 — 2026-03-20

---

## Кто мы

Мы помогаем технологическим продуктам и OSS-проектам выйти на рынок.
Специализация: продукты у которых нет сайта или есть слабый README без маркетинга.

**Не агентство полного цикла. Не SMM. Не "маркетинг для всех".**
Мы работаем только с tech/dev продуктами и знаем эту аудиторию изнутри.

---

## Что мы делаем

### Основной продукт: Landing Page Concept
1. Изучаем продукт, автора, конкурентов (5-6 файлов исследования)
2. Делаем дизайн-концепт лендинга: визуал, copy, структура
3. Приходим к фаундеру с готовым концептом — бесплатно или за символическую сумму
4. Если нравится → предлагаем реализацию (полная разработка сайта)

### Дополнительно: OSS Contributor Path
- Делаем fork репозитория
- Улучшаем README, описание, маркетинговые тексты
- Подаём PR как контрибьютор
- Вместе с PR предлагаем сделать полноценный сайт
- Это вход в проект без холодного питча — мы уже помогли

---

## Для кого

**Идеальный клиент:**
- OSS или indie tech продукт
- Есть рабочий продукт, есть GitHub (≥20 stars)
- Нет лендинга или лендинг — это просто README
- Фаундер — разработчик, не маркетолог (не знает как упаковать)
- Продукт имеет понятную ценность но плохо объяснён

**Где они живут:**
- GitHub (OSS без сайта — ищем по stars + no website link)
- Product Hunt (листинги с слабым описанием, нет сайта)
- HN Show HN посты (фаундеры сами пишут "нет нормального сайта")
- Reddit r/SideProject, r/IndieHackers

---

## Почему это работает

**Для фаундера:**
- Видит готовый результат до того как платит
- Не нужно объяснять что делать — мы сами исследуем
- Понимает ценность сразу (было vs стало)

**Для нас:**
- Нет холодных продаж без доказательства
- Каждый концепт = кейс для портфолио даже если не купят
- Концепт → деплой → апселл на следующий продукт фаундера

---

## Что мы НЕ делаем

- Не работаем с неопределёнными задачами ("сделайте нам маркетинг")
- Не беремся за продукты без понятного ценностного предложения
- Не продаём "стратегию" без реализации
- Не работаем с не-tech клиентами (не наша экспертиза)
