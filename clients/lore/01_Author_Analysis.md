## Product

**Lore** — lightweight desktop app (Electron), system tray resident.
- Summon: `Cmd+Shift+Space` → hover chat window
- Input: type a thought, task, or question
- Storage: local vector database (LanceDB)
- Processing: local LLM via Ollama
- Result: semantically searchable personal memory
- Privacy: zero cloud, no API keys, fully local

## Audience (from brief)

### Persona 1 — Primary: The Overloaded Dev
- Backend/full-stack, 25–35, startup or product company
- Constantly context-switching, needs <3-second thought dump
- Already knows Ollama. Comfortable with CLI and shortcuts.
- **Needs:** proof of speed (instant capture), proof of recall (RAG demo), download button
- **Trigger words:** "instant capture", "summon with a keystroke", "ask your own memory", "zero context switching"

### Persona 2 — Secondary: The Privacy Thinker
- Researcher/journalist/analyst, 28–45, works with sensitive info
- Won't build from source but will install a desktop app
- Uncomfortable with ChatGPT memory / cloud AI
- **Needs:** guarantee data never leaves machine, technical credibility (Ollama + LanceDB), open-source proof
- **Trigger words:** "zero cloud", "your data never leaves your machine", "no API keys", "fully private"

### Persona 3 — Aspirational (v2.0): Casual Thinker
- Non-technical, can't install Ollama
- Becomes viable when one-click setup ships
- Not targeted now, but don't alienate
