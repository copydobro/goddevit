## Starving Crowd — как конкуренты определяют аудиторию

**Obsidian:** "For everyone" → слишком широко. Девелоперы, студенты, писатели, исследователи — один продукт. Результат: steep learning curve, потому что нет ICP.

**Capacities:** Ближе всех — *"For you, not your team. Individuals who think for a living."* Но "knowledge workers" — это не starving crowd.

**Voicenotes:** Правильно сужается: мобильный пользователь который хочет говорить, не печатать. Но всё равно широко.

**Никто** не говорит прямо: *"Для разработчиков, которые теряют мысли при переключении контекста."* Это и есть настоящая starving crowd — боль конкретная, решаемая, платежеспособная.

## Бонусы и стекирование ценности

**Obsidian:** Плагины/темы как неявный бонус-стек — сотни вариантов. Но это не named bonuses — это скорее complexity overhead.

**Reflect:** Интеграции (Kindle, Calendar, Readwise) + Whisper + AI writing. Хороший стек, но не именован, не оценён в деньгах.

**Capacities:** "For you. Not your team." / "Funded by users, not VCs." / "Export everything. Always." — value stacking через values, не features. Нестандартно и работает.

**Вывод:** Никто не делает classic bonus stacking с именованными бонусами и денежными значениями. Пространство полностью пустое в категории.

## Гарантии — кто снимает риск

**Полный вакуум.** Ни один конкурент не предлагает явной гарантии. Ближайшие:
- Capacities: "Export everything. Always." — implicit no-lock-in guarantee
- Voicenotes: "100% self-funded" — implicit longevity signal
- Logseq: "Free forever for personal use" — lifetime price guarantee

**MIT License (Lore, Logseq, AFFiNE, Deta Surf)** — фактически самая сильная гарантия в категории: код существует вечно, форк всегда возможен. Но **никто это не продаёт как гарантию**.

---

## Scarcity & Urgency

**Практически ничего.** Closest:
- Capacities: "Founding member pricing" на раннем этапе
- Tana: waitlist в своё время → urgency через exclusivity
- Reflect: нет free tier → платный барьер как artificial qualifier

**Инсайт:** В этой категории scarcity строится не через количество мест, а через **Tana vacuum** (окно пока Tana уходит из PKM) и early adopter positioning.

## Grand Slam Offer для Lore

### Dream Outcome
> *"Every thought you'd normally lose — captured in 3 seconds, recalled in plain language, forever. Without opening an app, without a folder, without a server."*

### Value Equation — усиление по каждому рычагу

| Рычаг | Что есть | Как усилить |
|---|---|---|
| Dream Outcome | Capture + recall thoughts | Добавить emotional hook: *"the one that got away"* — боль потери идеи |
| Perceived Likelihood | 122 PH upvotes, 15 GitHub stars | Testimonial с конкретным результатом: *"saved 3 hours of lost context this week"* |
| Time Delay | 3 seconds capture ✅ | Подчеркнуть в headline + добавить *"recall in under 10 seconds"* |
| Effort & Sacrifice | Zero folders, zero API key ✅ | Добавить *"no setup, no onboarding, runs in 60 seconds after install"* |

### Grand Slam Offer Stack

```
Core: Lore app — instant hotkey capture + local AI recall
      Стоимость альтернативы: Reflect $10/mo = $120/yr

Bonus 1: "The Zero-Friction Install"
         Работает за 60 секунд после install
         Стоимость: 0 часов onboarding vs 4–8 часов для Obsidian

Bonus 2: "Local LLM Stack"
         Ollama + LanceDB, не нужен API ключ
         Стоимость альтернативы: OpenAI API = $5–20/мес

Bonus 3: "MIT License Guarantee"
         Код существует вечно, форк всегда возможен
         Не зависит от выживания компании

Bonus 4: "Zero-Cloud Privacy"
         Нет сервера, нет утечки, нет data breach
         Стоимость альтернативы: enterprise privacy software = $50+/мес

Total stack value: $120+/yr + privacy + time saved + lock-in freedom
Your investment: $0. MIT. Forever.
```

**Gap ratio:** аналог платных инструментов = $120–240/год. Lore = $0. Это **∞ : 1 value-to-price ratio** — сильнее любого конкурента в категории.

### Named Guarantee

> **"The Forever Guarantee"**
>
> MIT license means even if Lore disappears tomorrow, your data is local, the code is on GitHub, and you can run it forever. We literally cannot take it away from you.

Категорически сильнее любой "money-back guarantee": у конкурентов — 30 дней, у Lore — навсегда.

### Scarcity (реальная, не fake)

> *"Tana is abandoning PKM. 3,400 users are looking for a replacement right now."*

Реальный временной window. Messaging: *"The window where you can get in before everyone else is small."*
