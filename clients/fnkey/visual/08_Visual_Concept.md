# FnKey — Visual Concept & Design System

## Метафора

**Мир до / Мир после.**

Не до/после в смысле "плохо стало хорошо". В смысле двух миров с разной физикой.

- **Мир до** — мир клавиатуры. Гравитация есть. Трение есть. Мысль проходит через руки, пальцы, клавиши. Скорость ограничена пальцами (~50 WPM).
- **Мир после** — мир FnKey. Другая физика. Мысль = текст. Нет сопротивления. Скорость мышления = скорость вывода (~200 WPM).
- **Кнопка FN** — портал между двумя физическими системами. Не улучшение старого мира. Переход в другой.

---

## Визуальный стиль

**Black Terminal + Pixel 2D**

- Фон: `#0a0a0a` — почти чёрный, как экран терминала в покое
- Сетка: тонкая пиксельная, едва видимая. Не декорация — структура
- Акцент: `#F5A623` — янтарный. Единственный цвет. Один объект на экране
- Типографика: моноширинная (JetBrains Mono / IBM Plex Mono)
- Анимации: 8-bit / pixel-step движение, без плавных ease-кривых — только дискретные шаги
- Никаких скруглений, теней, градиентов ради красоты

## Hero-блок: Анимация с персонажами

### Сцена

Горизонтальная полоса посередине экрана — **земля**. Пиксельная, терминальная.
В центре на этой земле стоит **кнопка FN** — крупная, янтарная, светится пульсирующим свечением.
Кнопка — это буквально портал, граница двух миров. Левая половина экрана = до. Правая = после.

---

### Левый персонаж — Мир до

**Внешность:**
- Серый пиксельный человечек, 16×16 пикселей
- Голова чуть опущена, плечи ссутулены
- Руки перед собой — тянется к невидимой клавиатуре
- Цвет: `#555555` — серый, приглушённый

**Анимация движения:**
- Идёт влево от кнопки FN — медленно, 1-2 пикселя за кадр
- Цикл шагов: нога поднимается на 2px, опускается. Очень тяжело
- Каждые 3-4 шага — спотыкается, делает шаг назад на 1px
- Иногда останавливается, "думает" (над головой мигающий курсор `_`)

**Эффекты вокруг:**
- Вокруг него медленно падают одиночные буквы: `a`, `backspace↵`, `t`, `h`, `e`
- Буквы серые, некоторые зачёркнуты — ошибки
- Под ногами — след из точек, очень разреженный. Шёл долго, прошёл мало

**Физика:**
- Гравитация усиленная — он как будто тяжелее обычного
- Прыжок невозможен или очень низкий (2-3px от земли)

---

### Правый персонаж — Мир после

**Внешность:**
- Янтарный пиксельный человечек, тот же размер 16×16
- Голова поднята, руки раскинуты
- Цвет: `#F5A623` — янтарный, тот же акцент что и кнопка

**Анимация движения:**
- Движется вправо от кнопки — быстро, 4-6 пикселей за кадр
- Чередует: бег → прыжок → скольжение над землёй на 4px
- Прыжки высокие: 12-16px, дуга широкая
- Иногда делает двойной прыжок (wall jump от края)
- Никогда не спотыкается

**Эффекты вокруг:**
- Из него вылетают слова целиком: `"the"`, `"project"`, `"deadline"`
- Слова янтарные, моноширинные, летят вправо как снаряды
- Под ногами — плотный след из точек и слов. За секунду покрыл весь правый экран
- Лёгкое свечение вокруг тела — `box-shadow: 0 0 6px #F5A623`

**Физика:**
- Гравитация ослаблена — зависает в воздухе на лишний кадр
- Speed lines за спиной (4-6 горизонтальных пикселей, затухающих)

---

### Кнопка FN — центр сцены

- Размер: примерно 48×24px в пиксельном стиле
- Цвет: янтарный `#F5A623` с тёмной окантовкой
- Анимация покоя: пульсирует медленно (opacity 0.7 → 1.0 → 0.7, каждые 2 секунды)
- Над кнопкой: маленький мигающий курсор `▮`
- Под кнопкой: подпись `[fn]` моноширинным шрифтом, серая

**При hover пользователя на кнопку:**
- Свечение усиливается
- Левый персонаж замедляется ещё сильнее (как будто чувствует)
- Правый ускоряется

---

### Интерактивность (опционально, но сильно)

Пользователь **зажимает любую клавишу** на своём компьютере:
- Кнопка FN загорается на 100% brightness
- Правый персонаж переходит в режим turbo: скорость ×3, непрерывные прыжки
- Слова вылетают непрерывным потоком
- Левый персонаж замирает полностью

Пользователь **отпускает**:
- Кнопка возвращается в пульс
- Правый персонаж плавно (2 кадра) замедляется до обычного
- Левый снова медленно бредёт

Это демонстрирует продукт без единого слова объяснения.

---

### Земля и задний фон

- Земля: однопиксельная горизонтальная линия `#333333`
- Под землёй: темнее, `#070707`
- Фон: чистый `#0a0a0a`, без текстур
- Слева от центра: очень слабая сетка `#1a1a1a` — как терминальная разметка
- Справа от центра: те же клетки, но с редкими янтарными вспышками по узлам сетки

## Лейаут всего сайта: Принцип двух колонок

Каждая секция лендинга работает по одному принципу:

```
┌─────────────────┬────────────────────────────────────────┐
│   ЛЕВАЯ 20%     │            ПРАВАЯ 80%                  │
│                 │                                        │
│  название       │  весь контент секции                   │
│  секции         │  появляется мгновенно                  │
│                 │                                        │
│  печатается     │                                        │
│  по одной       │                                        │
│  букве          │                                        │
│  медленно_      │                                        │
└─────────────────┴────────────────────────────────────────┘
```

**Правило анимации:**
- Правая колонка: fade-in за 0.2s при появлении в viewport
- Левая колонка: typewriter эффект, 80ms на символ — намеренно медленнее
- Результат: правая сторона уже прочитана, пока левая ещё печатает название

**Это не баг — это демонстрация.** Пользователь телом чувствует разницу между двумя мирами.

## Цветовая система

| Роль | Цвет | Применение |
|---|---|---|
| Фон | `#0a0a0a` | Весь сайт |
| Текст основной | `#e0e0e0` | Правая колонка |
| Текст вторичный | `#555555` | Левая колонка, лейблы |
| Акцент / Мир после | `#F5A623` | Янтарный. Один. Везде |
| Линии | `#1f1f1f` | Сетка, разделители |
| Ошибки / Левый мир | `#444444` | Буквы вокруг левого персонажа |

## Типографика

| Уровень | Шрифт | Размер | Применение |
|---|---|---|---|
| Лейблы (левая колонка) | JetBrains Mono | 11px | Typewriter анимация |
| Заголовки (правая) | JetBrains Mono Bold | 32-48px | Instant появление |
| Тело | JetBrains Mono | 15px | Правая колонка |
| Кнопки | JetBrains Mono | 13px | Uppercase, letter-spacing: 2px |

Один шрифт. Всё моноширинное. Консистентность терминала.

## Секции лендинга через метафору

| Секция | Лейбл (левая, медленно) | Контент (правая, мгновенно) |
|---|---|---|
| Hero | — | Пиксельная анимация |
| Скорость | `speed_` | 50 WPM vs 230 WPM + графика |
| Как работает | `how_` | 3 шага: зажал → говоришь → готово |
| Приватность | `privacy_` | Микрофон умирает когда отпустил |
| Стриминг | `streaming_` | Текст идёт пока говоришь |
| Open source | `open_` | GPL-3.0, GitHub, бесплатно навсегда |
| Установка | `install_` | Одна команда |
| CTA финальный | `ready?_` | Кнопка скачать |
