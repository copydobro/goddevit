# Design System Documentation: Terminal Dark + Pixel Retro

## 1. Overview & Creative North Star

### The Creative North Star: "The Terminal Void"
Этот дизайн — гибрид эстетики **Obsidian** (техническая точность, чистое стекло) и **Trading Space** (ретро-игровой вайб, высокий контраст). Мы строим не просто сайт, а «экран терминала», где пиксельная игра является центральным артефактом.

**Принципы дизайна:**
*   **Monospace Sovereignty:** JetBrains Mono — единственный и основной шрифт. Он задает темп и структуру.
*   **The Black Void:** Истинный черный (#000) как база. Контент должен «светиться» на нем, как на старом мониторе.
*   **Optical Glass:** Использование тончайших линий и размытия (Obsidian-style) для структурирования интерфейса.

## 2. Colors: High-Voltage Contrast

Базовая палитра предельно функциональна.

| Role | Color | Usage |
| :--- | :--- | :--- |
| **Background** | `#000000` | Основной фон всей страницы. |
| **Surface**    | `#0A0A0A` | Вторичные блоки, "рецессия" терминала. |
| **Accent (Amber)** | `#F5C842` | Главные CTA, активные состояния. |
| **Accent (Green)**  | `#00FF41` | Статус "Online", успешные действия. |
| **Text (Primary)**  | `#FFFFFF` | Заголовки, читабельный текст. |
| **Text (Dim)**     | `#555555` | Метаданные, подзаголовки. |

---

## 3. Typography: Code as Content

*   **Font:** `JetBrains Mono`.
*   **Headlines**: Всегда Bold. Использовать `uppercase` для экстремально больших заголовков.
*   **Spacing**: Увеличенный `letter-spacing: 0.05em` для заголовков, чтобы добавить "технического" воздуха.

---

## 4. Components: The Terminal Interface

### Header (Obsidian Glass)
- **Background**: `rgba(10, 10, 10, 0.7)` с `backdrop-filter: blur(12px)`.
- **Border**: Нижняя граница `1px solid rgba(255, 255, 255, 0.08)`.
- **Items**: Тонкие, капсом, разделенные вертикальными черточками `|`.

### Buttons (High-Contrast Blocks)
- **Primary**: Янтарный (`#F5C842`) фон, черный текст. Никаких скруглений (`border-radius: 0`).
- **Secondary**: Белая обводка 1px, прозрачный фон.

### The Game Console
- Игра вложена в контейнер с имитацией рамки монитора или просто выделена свечением (box-shadow: 0 0 40px rgba(245, 200, 66, 0.1)).

---

## 5. Do’s and Don’ts

### Do
- Используйте `line-height` 1.6 для читабельности моноширинного текста.
- Используйте `|` и `_` как элементы декора.
- Сохраняйте пиксельную четкость (`image-rendering: pixelated`) для всех графических элементов.

### Don’t
- Не используйте тени (кроме свечения).
- Не используйте градиенты (стиль должен быть плоским и "жестким").
- Не используйте скругления углов. Терминал — это острые углы.
