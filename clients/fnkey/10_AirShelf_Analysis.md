## Product — What It Does

AirShelf позиционируется как нейтральная инфраструктура между ритейлерами и AI-агентами. Ключевое сообщение: не просто попасть в AI-рекомендации, но и закрывать транзакции через них.

---

## Features

| Feature | Description |
|---------|-------------|
| Agentic Commerce Protocol (ACP) | Стандарт взаимодействия с AI-агентами |
| Universal Commerce Protocol (UCP) | Универсальный протокол коммерции |
| AI Shelf Share | Трекинг и аналитика присутствия в AI |
| AI-ready product catalog API | JSON-LD + векторы для AI-индексации |
| Native agentic checkout | Нативная оплата через агентов |
| Real-time inventory sync | Синхронизация остатков в реальном времени |
| Multi-platform integration | Shopify, WooCommerce, BigCommerce |
| Agent Experience (AX) scoring | Оценка качества взаимодействия с агентом |
| Answer Engine Optimization (AEO) | Оптимизация под ответные движки |
| Generative Engine Optimization (GEO) | Оптимизация под генеративные движки |
